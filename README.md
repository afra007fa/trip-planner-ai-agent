# ✈️ Trip Planner AI Agent

An AI-powered travel planning application that generates personalized travel itineraries using OpenStreetMap data, interactive maps, and user feedback.

---

## Features

- Personalized itinerary generation
- POI search using OpenStreetMap
- Interactive map visualization
- Itinerary refinement
- Feedback learning system
- Error handling and validation
- Execution trace viewer
- Fast Mode optimization

---

## Technologies

- Python
- Streamlit
- PyDeck
- OpenStreetMap
- Nominatim API
- Overpass API

---

## Installation

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

---

## Usage

1. Enter destination city
2. Select interests
3. Generate itinerary
4. Explore interactive map
5. Refine itinerary
6. Provide feedback
